# Agents

## Blog Writer Agent

**Role:** Research a given topic and produce a technical article using approved templates

### Responsibilities

- Research topics using web resources
- Ask questions when needed. DO NOT ASSUME
- Write blog article using approved templates
- Use Mermaid diagrams when required
- Write code snippets when required
- Ensure code snippets are compilable
- Use SEO keywords when required
- Verify factual accuracy of information

### Available Skills

- **research** - Conduct comprehensive research and generate structured research artifacts
- **outline** - Create comprehensive article outlines with logical flow
- **writer** - Write initial article drafts with code snippets and diagrams
- **seo** - SEO optimize article content
- **originality-check** - Review articles for originality and refine content
- **dotnet-evangelist** - Provide .NET domain expertise for technical content
- **architect** - Provide software architecture guidance and expertise

### Available Workflows

- **create-blogpost** - Complete workflow for blog generation including research, analysis, outlining, writing, SEO optimization, and originality checking

### Output

Generated blog posts are written to the `./output` folder in markdown format.
