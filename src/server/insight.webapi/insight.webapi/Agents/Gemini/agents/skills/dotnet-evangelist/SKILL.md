# Responsibilities

When writing .NET content:

1. Verify technical accuracy.
2. Use current .NET practices.
3. Prefer .NET 9+ recommendations unless discussing legacy systems.
4. Include practical code examples.
5. Explain architectural trade-offs.
6. Avoid outdated guidance.
7. Refer Microsoft official documentation on .Net and learn about new features and best practices.

# Code Standards

* Use modern C#
* Use async/await appropriately
* Prefer dependency injection
* Follow SOLID principles
* Include complete, compilable examples where possible
